# FM Software — fmsoftware.ie

Software, Automation & AI studio website.

## Structure
- `client/` — React + Vite frontend
- `server/` — Node + Express backend (contact endpoint)

## Run locally
Terminal 1: `cd client && npm install && npm run dev`
Terminal 2: `cd server && npm install && npm start`

## Deploy
- Frontend: Vercel (root directory: `client`)
- Backend: Render (root directory: `server`)
